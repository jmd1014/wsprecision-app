# 현재 기능 맵 (Feature Map)

우성정밀 ERP/MES Streamlit 앱의 현재 기능을 **실제 운영 / 보조 진단** 두 축으로 분류.
*Master Stabilization Phase* (마스터 안정화 단계) 기준.

---

## A. 실제 운영 기능

| # | 화면 | 우선순위 | 역할 | 현재 문제점 |
|---|---|---|---|---|
| A1 | 🏠 홈 | 일상 | 마스터 카운트 / TOP10 / 로드맵 | (안정) |
| A2 | 📥 수주 관리 | 일상 (핵심) | Excel/PDF 파싱 + 수기 입력 → sales_orders | UI 폼이 많음. 분기 제품 수주는 별 검증 없이 통과 |
| A3 | 🧱 제품/BOM 관리 | 정비 (마스터) | 제품 마스터 편집 + 휴면 처리 + 거래처 편집 | "마스터 관리" 페이지 안에 7탭 — 분기점 많음 |
| A4 | 📦 자재 관리 | 정비 (마스터) | 자재 마스터 편집 | 마스터 관리 탭으로 묶여 있음 (분리 안 됨) |
| A5 | 🔗 BOM 정비 | 정비 (핵심) | 1제품 BOM 자재행/공정행 입력 + 분기 복사 | LOT 단가 입력은 제거됨. 검색 결과 표시 OK |
| A6 | 💰 원가 확인 | 검증 | BOM 원가/판매가/마진 비교 + 매입 단가 조회 | 6탭 중 2탭이 "참고" 라벨 — 이상치/BOM재산정 |
| A7 | 📋 발주서 작성 | 일상 (핵심) | 거래처/품목 → 발주번호 자동 (DB 함수) → 양식 다운로드 | 동시성 안전 (Migration 005) |
| A8 | 📊 생산 준비 | 일상 | BOM × 수주 → 자재 필요량 산출 | 단순 자재 산출만, 공정 일정 X |
