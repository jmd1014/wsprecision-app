# 생산 영역 추가 로드맵 (수주-소재입고-**생산**-납품)

현재 흐름에서 생산이 빠져 있음:

```
[현재]   수주 ✅ → 자재 필요량 ✅ → 발주 ✅ → (소재입고 ❌) → (생산 ❌) → 납품 ✅
[목표]   수주 → 자재 필요량 → 발주 → 소재입고 → 생산 투입/완료 → 납품
```

## 추가 타이밍: 3단계 (의존성 순서)

### Phase A — 소재 입고 ✅ **완료 (2026-07-03, Migration 017)**
- 구매/발주 → 발주 이력 → 발주 선택 → 📦 입고 처리 섹션
- 입고 시: `inventory_transactions` RECEIPT INSERT + PO 상태 자동 (RECEIVED/PARTIAL)
- 자재 매핑: 라인별 최초 1회 지정 → `purchase_order_items.material_id` 재사용
- `material_stock` = 기초재고(import 스냅샷) + 원장 누적 → 생산 준비 실재고 연동됨
- 입고 현황은 `po_item_receipt_v` (원장 집계 — 저장 컬럼 없음)

### Phase B — 생산 보고 ✅ **완료 (2026-07-03, Migration 018)**
- 🏭 생산 보고 페이지 (사이드바 메인 그룹)
- 생산 보고 입력: 제품 검색 → 양품/불량/일자/교대/비고
  → `production_log` INSERT (product_id 포함)
- BOM 기준 자재 자동 차감: PROD_INPUT (음수) — 차감 미리보기 + 재고 부족 경고
- 제품 완성 재고: PROD_OUTPUT (양품만) → `product_stock_v` view
- BOM 미매핑 제품은 차감 없이 기록만 (경고 표시)
- 생산 이력 탭: 보고 목록 + 불량률 + 제품 재고 현황

### Phase C — 납품-생산 연결 ✅ **완료 (2026-07-03, Migration 019)**
- 납품 등록 시 제품 재고 차감 (ISSUE 원장, product_id 매핑 라인)
- LOT 추적: 생산 보고 LOT 자동 제안 (`LOT-YYMMDD-품번`)
  → PROD_INPUT/OUTPUT 원장에 기록 → 납품 출고 LOT 로 연결
- 역추적 화면: 생산 보고 → 🔎 역추적 탭
  · LOT 기준: 자재 투입 → 생산 완성 → 납품 출고 타임라인 + LOT 잔량
  · 제품 기준: 전체 원장 이력 (LOT 유무 무관)
- view: `lot_trace_v` / `product_trace_v`
- 실원가 (`product_actual_cost_v`) 는 production_log 누적과 함께 자동 활성

### Phase D — MES 실적 업로드 ✅ **완료 (2026-07-06, Migration 020)**
- 생산 보고 → 📥 MES 업로드 탭: 일간 생산보고서 엑셀(.xls=HTML) 업로드
  → 파싱 → 품번 매칭 → 검수 그리드 → `production_log` 저장 (source=MES_UPLOAD)
- ⚠️ **재고 연동 없음** — 공정별 실적 raw 전용. 완성 재고(PROD_OUTPUT) 연결은
  방식 확정 전까지 보류 (수기 생산 보고로 별도 진행)
- 중복 방지: 같은 날짜+교대 재업로드 시 교체(권장)/추가 선택
- 생산 이력: 소스 필터(수기/MES) + 설비별/품번·공정별 집계
- E2E 검증 완료: 실샘플 68행 (2026-07-03, 생산 5,451/불량 9) 저장·조회 확인

## 전환 시점과의 관계

```
기능 테스트 (지금) ──→ Phase A (소재 입고) ──→ 실무 전환 가능 시점 ①
                                │                  (수주~발주~입고~납품 흐름 완성)
                                ▼
                          Phase B (생산 보고) ──→ 실무 전환 가능 시점 ② (권장)
                                │                  (재고 자동 차감까지)
                                ▼
                          Phase C (LOT/실원가) ──→ 완전체
```

**권장**: Phase A 까지 완성 후 실무 전환 (시점 ①). 생산 보고(B)는 운영하면서 추가해도 흐름이 안 깨짐 — 생산 전에는 재고 차감을 수동 조정(ADJUSTMENT)으로 보완 가능.
